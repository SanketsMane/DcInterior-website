## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd dcinteriors-website
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📦 Build & Deployment

### For Static Hosting (Recommended)
```bash
npm run build
```

The `out` folder will contain the static files ready for deployment to any static hosting service.

### Deployment Options

1. **Vercel** (Recommended)
   - Connect your GitHub repository to Vercel
   - Automatic deployments on push

2. **Netlify**
   - Drag and drop the `out` folder
   - Or connect via Git

3. **GitHub Pages**
   - Push the `out` folder contents to gh-pages branch

4. **Any Static Hosting**
   - Upload the `out` folder contents
