# 🏛 ModernSpace Studio Website

A fully responsive and modern **Interior Design Website** built using **Next.js**, **Tailwind CSS**, and **Shadcn UI**.  
Designed with a sleek and elegant layout, this website showcases services, projects, and client testimonials to deliver a premium brand experience for **ModernSpace Studio**.

## 📌 Features

* **Home Page**
  * Hero Section with striking visuals
  * Services Section with detailed service cards
  * Featured Projects with interactive hover effects
  * Client Testimonials showcasing experiences
  * Call-to-Action (CTA) section for immediate engagement

* **About Page**
  * Brand story and philosophy
  * Team Section highlighting our expert designers

* **Services Page**
  * Comprehensive Services Card List
  * Design Process section for transparency and clarity
  * CTA to encourage project inquiries

* **Portfolio Page**
  * Grid layout of all completed projects with modern styling

* **Contact Page**
  * Contact form for inquiries
  * Business contact details and quick links

* **UI & UX Enhancements**
  * Sticky navigation bar for smooth browsing
  * Minimalist and elegant footer
  * Fully responsive design for all devices
  * Built using **Shadcn UI components** for a polished look

## 🛠 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **UI Components:** [Shadcn UI](https://ui.shadcn.com/)


### 2️⃣ Install Dependencies

```bash
pnpm install
```

### 3️⃣ Run the Development Server

```bash
pnpm run dev
```

Website will be live at: [http://localhost:3000](http://localhost:3000)
